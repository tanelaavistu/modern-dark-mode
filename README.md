# Modern Dark Mode

A sleek, efficient dark/light mode switcher plugin for WordPress websites.


## Features

- Easy Toggle: A simple button lets users switch modes instantaneously.
- System Theme: Switch theme automatically by native theme.
- Performance-Oriented: No plugin needed!

## Installation

1. Set the head script
2. Set the body script
3. Add CSS to your custom CSS.
4. Add buttons as you like.
